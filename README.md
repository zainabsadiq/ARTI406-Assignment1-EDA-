# Sleep Health and Lifestyle Dataset

## Dataset Source
This dataset is a public lifestyle and health dataset commonly shared on open data platforms such as Kaggle for educational analysis and machine learning practice.

## Dataset Size
- **Rows:** 374
- **Columns:** 13

## Features (Columns)
| Column | Description |
|---|---|
| Person ID | Unique identifier for each person |
| Gender | Gender of the individual |
| Age | Age of the individual |
| Occupation | Profession or job title |
| Sleep Duration | Average number of hours slept per day |
| Quality of Sleep | Numeric score representing sleep quality |
| Physical Activity Level | Numeric indicator of activity level |
| Stress Level | Numeric score representing stress level |
| BMI Category | Body Mass Index category |
| Blood Pressure | Blood pressure recorded as systolic/diastolic |
| Heart Rate | Resting heart rate |
| Daily Steps | Number of steps taken per day |
| Sleep Disorder | Reported sleep disorder, if any |

## Purpose of Using This Dataset
This dataset is used to study how lifestyle and health-related factors are associated with sleep duration, sleep quality, and sleep disorders. It is suitable for:
- Exploratory Data Analysis (EDA)
- Data cleaning practice
- Data visualization
- Health trend analysis
- Predictive modeling

## Dataset File
- **File name:** `Sleep_Health_and_Lifestyle_Dataset.csv`
- **Format:** CSV
